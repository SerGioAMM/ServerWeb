Factorización

Descripción

Esta herramienta hecha en genially esta diseñada para ayudar a los estudiantes a aprender de manera interactiva y simple a factorizar.




Tecnologías utilizadas:

  • Genially

Proceso de Desarrollo

#1

Haciendole preguntas directamente al estudiante, logramos crear una herramienta muy amigable e intuitiva.

#2

Agregamos las diferentes variantes para que el estudiante pueda elegir la que mejor se adapte a sus necesidades.

#3

Tiene ejemplos con cada paso explicado para que el estudiante sepa que es lo que está pasando.
